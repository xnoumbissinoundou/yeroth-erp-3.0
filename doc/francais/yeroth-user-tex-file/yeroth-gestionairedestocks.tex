Un utilisateur de \yeren avec le \role \gestionairedestocks a acc\`es
aux fonctionnalit\'es suivantes:
\begin{enumerate}[1)]
	\item alertes
	\item gestion des achats
	\item gestion des stocks (voir le Tableau~\ref{tab:taches-fonctions}).\\
\end{enumerate}
